Apply BEFORE finding the critical range:

| Grade | Modifier |
|---|---|
| Very Easy | skill x 2 |
| Easy | skill x 1.5 |
| Standard | -- |
| Hard | skill x 2/3 |
| Formidable | skill x 1/2 |
| Herculean | skill x 1/5 |
| Hopeless | cannot attempt |

When multiple penalties apply, use the most severe only.
