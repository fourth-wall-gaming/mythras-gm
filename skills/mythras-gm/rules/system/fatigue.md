Each failed exertion roll (Athletics/Brawn/Endurance) adds a level:

Fresh -> Winded (skills Hard) -> Tired (Hard, -1m move) -> Wearied (Formidable, -2m, -2 init) -> Exhausted (Formidable, move halved, -4 init, -1 AP) -> Debilitated (Herculean, -6 init, -2 AP) -> Incapacitated -> Semi-Conscious -> Comatose -> Dead.

Recovery = one period per Healing Rate (15min for Winded, up to 24h+ for deeper levels). Settings may add environmental Fatigue rules (e.g. exhausting flight conditions) -- see the campaign's lore.
