When circumstances affect the chance of success, apply a Difficulty Grade. Use the **default** (scaling) modifier OR the **Simplified** (flat %) modifier consistently throughout the game.

| Difficulty Grade | Skill Modifier | Simplified Modifier |
|:--|:--|:--|
| Automatic | No roll needed | No roll needed |
| Very Easy | x2 skill value | +40% |
| Easy | skill + half again | +20% |
| Standard | No adjustment | None |
| Hard | skill x 2/3 (reduce by 1/3) | -20% |
| Formidable | skill x 1/2 | -40% |
| Herculean | skill x 1/5 | -80% |
| Hopeless | No attempt possible | No attempt possible |

Where multiple penalties apply, the **hardest Difficulty Grade takes precedence** (they do not stack).

Outside combat, the GM decides which grade fits the situation based on character capabilities, perceived difficulty, and dramatic tension.
