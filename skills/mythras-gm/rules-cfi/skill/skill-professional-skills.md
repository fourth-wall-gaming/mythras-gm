Professional Skills differ between characters, reflecting specialist training from culture and class. They function exactly like Standard Skills.

**Magical skills** (Arcane Casting, Arcane Knowledge, Channel, Devotion) are Professional Skills; see the Magic chapter for full details.

| Skill | Base Value | Notes |
|:--|:--|:--|
| Acting | CHA x2 | Impersonation and portraying a different character convincingly; synergizes with Disguise and Deceit |
| Acrobatics | STR+DEX | Balance, gymnastics, juggling, tumbling; mitigate fall damage (halved on success); substitute for Evade (no prone result); basis for Rogue Vaulting |
| Animal Handling | POW+CHA | Calm hostile animals or train domestic ones; success drives off wild animal; +2 grades harder if hungry or has young; offering food may ease by 1-2 grades |
| Arcane Casting | INT+POW | Cast memorized arcane spells; higher skill improves Intensity of spells cast |
| Arcane Knowledge | INT x2 | Accumulated spell knowledge; learn/copy spells, create scrolls, identify casting; determines starting spells; can be taken without practicing Arcane Magic |
| Art | POW+CHA | Specialized art form (e.g. Art (Painting)); each form advanced separately |
| Bureaucracy | INT x2 | Administrative procedures, records, navigating red tape; cannot exceed Language/Literacy score in the relevant working language |
| Channel | INT+CHA | Call upon divine power; higher skill improves Intensity of divine spells cast |
| Commerce | INT+CHA | Evaluate and trade goods; find buyers for stolen goods; Opposed by Commerce or Willpower |
| Courtesy | INT+CHA | Correct social behavior, modes of address, titles, formal rituals |
| Craft | DEX+INT | Specialized craft form (e.g. Craft (Pottery)); each craft advanced separately |
| Culture | INT x2 | Foreign societies (specify one per skill); mechanically identical to Customs |
| Devotion | POW+CHA | Cleric's commitment to deity; governs Magnitude of Divine spells; can indicate standing in religious org; can be taken without practicing Divine Magic |
| Disguise | INT+CHA | Convincing disguise using costumes, cosmetics, wigs; lacking time or materials increases Difficulty Grade |
| Engineering | INT x2 | Design/build large structures (houses, bridges, siege engines); assess structural integrity |
| Gambling | INT+POW | Games of chance; assess odds; detect cheating; Opposed or unopposed depending on stakes |
| Healing | INT+POW | In-depth medical knowledge (herbs, drugs, sutures, bone-setting); requires appropriate resources |
| Intimidation | INT+CHA | Threaten someone into unwilling action; Opposed by Willpower (or Endurance if torture used) |
| Language | INT+CHA | Foreign language fluency; not rolled directly -- static fluency cap (see Linguistic Fluency table); includes literacy; each language separate |
| Language (Thieves' Cant) | INT+CHA | Rogues only; requires Thieves' Guild Oath; slang for illicit discussions; Guild members start at Base+40%; others start at Base |
| Lockpicking | DEX x2 | Open mechanical locks without the key; lever open bolts/bars without damage; reverse-pick to lock without a key |
| Lore | INT x2 | Specific body of knowledge (e.g. Lore (Alchemy), Lore (Monsters)); each topic separate |
| Mechanisms | DEX+INT | Assemble/disassemble mechanical devices and traps; distinct from Lockpicking |
| Musicianship | DEX+CHA | Play musical instruments; each skill covers a group of similar instruments (e.g. Flutes covers reed pipe, flute, panpipes, recorder) |
| Navigation | INT+POW | Accurate direction-finding (stars, landmarks, seawater); each environment/region separate; used in unfamiliar territory |
| Oratory | POW+CHA | Deliver persuasive speech to large groups; used by commanders and politicians; replaces Influence for mass persuasion |
| Seamanship | INT+CON | Large waterborne vessels (sail or oars); maintenance, safe anchorage, weather assessment |
| Seduction | INT+CHA | Romantic/sexual persuasion using verbal and non-verbal signals; takes hours to weeks; Opposed by Willpower |
| Sleight | DEX+CHA | Palm/conceal small objects, pick pockets, cut purses, visual distraction |
| Streetwise | POW+CHA | Knowledge of settlements: dangerous areas, legal/illegal services, social contacts |
| Survival | CON+POW | Wilderness survival: foraging, fire, shelter; roll once per day when separated from equipment or in bad conditions |
| Track | INT+CON | Follow game or quarry by signs (footprints, disturbed foliage); periodic rolls as conditions change; Opposed by Conceal |
