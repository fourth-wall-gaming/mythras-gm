Standard Skills are common to every character. Base value is derived from two Characteristics (or one x2). Some include a flat bonus.

| Skill | Base Value | Notes |
|:--|:--|:--|
| Athletics | STR+DEX | Climbing, jumping, throwing, running |
| Boating | STR+CON | Small craft (boats, canoes, rafts) on rivers, lakes, inshore; large vessels use Seamanship |
| Brawn | STR+SIZ | Applied might: lifting, breaking down doors, strength contests |
| Conceal | DEX+POW | Hiding objects (not self); counterpart to Stealth |
| Customs | INT x2+40 | Own community's social codes, rites, taboos |
| Dance | DEX+CHA | Rhythmic movement for recreation or ritual |
| Deceit | INT+CHA | Lying, misleading, bluffing, feigning emotions; countered by Insight |
| Drive | DEX+POW | Wheeled/drawn vehicles (carts, carriages, chariots, sleds) |
| Endurance | CON x2 | Physical stress, pain, fatigue; resisting injury effects, poison, disease |
| Evade | DEX x2 | Dodge ranged attacks, traps, changing combat distance; some magic resistance |
| First Aid | DEX+INT | Treat minor injuries; heals 1d3 damage; only once per specific injury |
| Influence | CHA x2 | Personal persuasion; Opposed by Perception, Willpower, or Influence |
| Insight | INT+POW | Read motives and body language; detect lies; Opposed by Deceit |
| Locale | INT x2 | Local flora, fauna, terrain, weather in home region; harder in unfamiliar areas |
| Native Tongue | INT+CHA+40 | Own language fluency; not rolled directly -- caps verbal interaction; can interpret related dialects (at least one grade harder) |
| Perception | INT+POW | Passive observation and focused detection; darkness/noise affects grade |
| Ride | DEX+POW | Control ridden creatures; unfamiliar species +1 grade harder; different medium +2 grades harder |
| Sing | POW+CHA | Carrying a tune, rhythm, key, lyrics |
| Stealth | DEX+INT | Hiding and moving silently; cover/darkness improve grade; armor noise may worsen it |
| Swim | STR+CON | Negotiate water; rough seas/currents reduce grade |
| Unarmed | STR+DEX | Brawling and wrestling; Critical/Fumble effects use combat rules |
| Willpower | POW x2 | Mental resilience, concentration, resisting magic; Critical = immune to further mental influence for that encounter |
