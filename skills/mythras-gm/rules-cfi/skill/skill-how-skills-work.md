A skill has a **Base Value** built from two Characteristics (or one Characteristic x2). There is no upper limit.

**Resolution**: Roll 1d100.
- <= skill value: **Success**
- > skill value: **Failure**
- 01-05: always a Success regardless of skill value
- 96-00: always a Failure regardless of skill value

**Critical Success**: roll <= 1/10 of the skill value (round up). Modifiers apply before calculating the critical range (e.g. skill doubled from 40% to 80% means critical on 08 or less).

**Fumble**: roll of 99 or 00. If skill > 100%, Fumble only on 00.

**Automatic Success**: when a task is straightforward enough that success is assured given time and relevant skill, no roll is needed (e.g. gentle canter across open ground, a craftsman carving unhurriedly, an undisturbed rogue picking a simple lock).

**Reattempting a failed roll**: the GM may allow a single follow-up attempt at one grade harder. If that also fails, no further attempts may be made for the foreseeable future.
