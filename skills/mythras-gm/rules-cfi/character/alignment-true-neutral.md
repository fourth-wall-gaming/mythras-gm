**True Neutral** -- Base Level: INT + POW + 30.

A character who would otherwise choose Neutral for BOTH their Ethical Code and their Moral Code must instead disregard both and take True Neutral as a single, unified Alignment.

**True Neutral** -- Choose one or two defining elements:
Ambivalent; Cautious; Committed to others by personal relationships; Flexible; Freedom of action; Open-minded; Pragmatic; Respectful of Nature; Strives for balance.
