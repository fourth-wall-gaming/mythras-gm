Bonus Skill Points are NOT cumulative -- a Rank 2 character is built with 300 points total, not 300 + 100.

| Rank (Levels) | Age Bonus | Bonus Skill Points | Max Per Skill | Avg Skill Level | Advancement Requirement |
|:---|:---:|:---:|:---:|:---:|:---|
| Rank 1 (1-4): Fresh | By species | 100 | +10 | 40-60 | Any 5 Class Skills at 40% |
| Rank 2 (5-9): Low Level | +1d6 yrs | 300 | +30 | 65-75 | Any 5 Class Skills at 70% |
| Rank 3 (10-14): Mid-Level | +1d6+5 yrs | 500 | +50 | 85-95 | Any 4 Class Skills at 90% |
| Rank 4 (15-19): High Level | +1d6+10 yrs | 700 | +70 | 105-115 | Any 3 Class Skills at 110% |
| Rank 5 (20+): Epic Level | +1d6+20 yrs | 900 | +90 | 125-135 | Any 2 Class Skills at 130% |

All characters gain +1 Action Point and +1 Hit Point per location at Ranks 2 and 4. All characters gain +1 Luck Point per new Rank. NPCs built as careers have no special Abilities (including extra Luck Points). NPC starting Rank is 1 by default.
