**Alignment** has two parts: an **Ethical Code** (Law/Neutral/Chaos) and a **Moral Code** (Good/Neutral/Evil).

**Construction:**
1. Choose one Ethical Code value: Lawful, Neutral, or Chaotic.
2. Choose one Moral Code value: Good, Neutral, or Evil.
3. Each starts at Base Level = INT + POW + 30.
4. Choose one or two defining elements per Code (Neutral allows only one defining element).
5. Exception: if you would choose Neutral for BOTH codes, disregard both and instead take **True Neutral** as a single Alignment.

**Example:** Chaotic (Disorganized and Greedy) 55%, and Good (Helpful and Charitable) 55%.

A character's Alignment score may drop during play if the character continually disregards their chosen morality/ethics. If it drops below the Base Level, the Alignment can shift to an opposing value (e.g., Good to Neutral or Evil). The GM determines the new Alignment. A change may result in losing Class Abilities.
