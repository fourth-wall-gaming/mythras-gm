Add **STR + SIZ** and look up the Damage Modifier below. This die is added to (or subtracted from) all physical damage rolls. If a negative modifier reduces damage to 0 or below, no damage is dealt.

| STR+SIZ | Damage Modifier |
|:-------:|:---------------:|
| 5 or less | -1d8 |
| 6-10 | -1d6 |
| 11-15 | -1d4 |
| 16-20 | -1d2 |
| 21-25 | +0 |
| 26-30 | +1d2 |
| 31-35 | +1d4 |
| 36-40 | +1d6 |
| 41-45 | +1d8 |
| 46-50 | +1d10 |
| 51-60 | +1d12 |
| 61-70 | +2d6 |
| 71-80 | +1d8+1d6 |
| 81-90 | +2d8 |
| 91-100 | +1d10+1d8 |
| 101-110 | +2d10 |
| 111-120 | +2d10+1d2 |
| 121-130 | +2d10+1d4 |
| Each +10 | Continue progression |
