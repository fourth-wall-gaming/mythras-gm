**Passion Table -- Starting Percentage by Object Type**

| Object of Passion | Starting Percentage |
| :-- | :-- |
| A person (romantic or familial) | POW + CHA |
| An organization or group of people | POW + INT |
| A race or species | POW x 2 |
| A place | POW + INT |
| An object or substance | POW x 2 |
| A personality trait | POW + CHA |

Starting bonuses (+40/+30/+20) are added ON TOP of the base value from this table.
