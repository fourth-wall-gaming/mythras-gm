Every character has all Standard Skills at their base level. Skill % = the formula shown. Customs and Native Tongue include a static +40% bonus.

| Skill | Base Percentage |
|:------|:---------------:|
| Athletics | STR+DEX |
| Boating | STR+CON |
| Brawn | STR+SIZ |
| Conceal | DEX+POW |
| Customs | INT x2 +40* |
| Dance | DEX+CHA |
| Deceit | INT+CHA |
| Drive | DEX+POW |
| Endurance | CON x2 |
| Evade | DEX x2 |
| First Aid | INT+DEX |
| Influence | CHA x2 |
| Insight | INT+POW |
| Locale | INT x2 |
| Native Tongue | INT+CHA +40* |
| Perception | INT+POW |
| Ride | DEX+POW |
| Sing | CHA+POW |
| Stealth | DEX+INT |
| Swim | STR+CON |
| Unarmed | STR+DEX |
| Willpower | POW x2 |

Combat Skill (STR+DEX) is also a Standard Skill, learned through Class.

*Humans speak Common at +40%; each demi-human race may also speak Common at +40% in addition to their Native Tongue at +40%.
