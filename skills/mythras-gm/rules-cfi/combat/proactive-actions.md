Each costs 1 Action Point unless noted. Taken on the character's own Turn.

| Action | Notes |
|---|---|
| **Attack** | Strike with melee or ranged weapon. |
| **Brace** | Firm stance: SIZ treated as 50% larger vs. Knockback/Leaping; SIZ doubled vs. Bash. |
| **Cast Magic** | Cast spell or invoke racial/class ability. Complex magic may take multiple Actions. |
| **Change Range** | Close in on or retreat from an opponent. |
| **Charge** | Move at Run or Sprint into melee; see Charging rules. |
| **Delay** | Hold Action Points for later Reactive use; lost if unused before next Turn. |
| **Dither** | Do nothing (default if no other Action chosen). |
| **Hold Magic** | Maintain a completed but un-released spell; required each Turn to keep it ready. |
| **Hustle** | Run (3x Move) or Sprint (5x Move); no other Proactive Actions this Turn. |
| **Mount** | Mount or dismount a riding beast or vehicle; may take multiple Turns. |
| **Move** | Move up to Base Movement Rate; may be used multiple times per Round. |
| **Outmaneuver** | Group Opposed Evade roll vs. multiple opponents; those who fail cannot attack this Round. |
| **Ready** | Draw, sheath, withdraw, or reload weapon or object. Picking up dropped object costs 2 Actions. |
| **Regain Footing** | Auto if unengaged; Opposed Brawn/Athletics vs. opponent if Engaged. |
| **Struggle** | Attempt to break free from Grapple or similar Special Effect. |
| **Take Cover** | Duck behind nearby cover; does not leave you prone. Requires available cover. |
