| Situation | Difficulty Grade |
|---|---|
| Light wind* | Hard |
| Moderate wind* | Formidable |
| Strong wind* | Herculean |
| Gale, storm, or worse* | Hopeless |
| Target is Running | Hard |
| Target is Sprinting | Formidable |
| Target obscured by mist or partial darkness | Hard |
| Target obscured by thick smoke or darkness | Formidable |
| Target completely obscured | Herculean |
| Target completely invisible** | Impossible |
| Target prone | Formidable |
| Attacker prone*** | Herculean |
| Attacker on unstable ground | Hard |

*Applies only to weapons/ammunition vulnerable to crosswinds.
**If general location is known, treat as Herculean instead.
***Penalty negated if using a crossbow from a prepared position.

**Aiming:** Spend a full Round aiming to reduce one Situational Modifier by one grade. No benefit from additional Rounds spent aiming.

**Firing into a crowd:** Hard at crowd edge, Formidable to shoot through it. On failure but would-have-succeeded-without-penalty, adjacent bystander is hit instead (may Evade/Parry normally but suffers no Special Effects).
