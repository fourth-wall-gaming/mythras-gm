**Base:** All characters begin with 2 Action Points per Round.

Additional Action Points are granted by class Rank advancement -- see each class Rank table. Each class table includes an **Action Points** column showing the cumulative bonus at that Rank.

**Example fighter Rank bonuses:**
- Rank 0-1: +0 (total 2)
- Rank 2: +1 (total 3)
- Rank 3: +1 (total 3, from prior)
- Rank 4: +2 (total 4)
- Rank 5: +2 (total 4, from prior)

Proactive Actions may only be spent on the character's own Turn. Reactive Actions (Parry, Counter Spell, Evade, Interrupt) are spent in response to incoming threats. Unused Action Points are lost at the end of the Round.
