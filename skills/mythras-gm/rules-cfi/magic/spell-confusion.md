**Confusion** (Enchantment (Charm))

- **Rank:** Cleric 4, Mage 3
- **Casting Time:** 2 Actions
- **Duration:** 1 Minute/Intensity
- **Cost:** 3+1/additional Intensity
- **Range:** 120 ft (Cleric version 250 ft)
- **Area:** 30 ft. Radius
- **Resist:** Willpower (Cleric version: Hard Willpower)

1d4 creatures SIZ 1-20 per Intensity (Cleric: 1d3+1 per Intensity, no maximum). Closest to caster affected first. Roll d100 each Round for each confused creature:

| d100 | Effect |
| :-: | :-- |
| 01-10 | Attack caster with ranged or melee weapons |
| 11-50 | Stand confused (Dither) |
| 51-70 | Attack nearest enemy |
| 71-80 | Attack nearest ally |
| 81-90 | Wander off at Walk speed (or Dither if engaged) |
| 91-00 | Act normally |

Each attack on a confused creature allows it another Resistance roll. Large (SIZ 21-40): Easy resistance; Huge (SIZ 41+): Very Easy.

| Intensity | Cost | Duration | Mage Creatures | Cleric Creatures |

| 2 | 4 | 2 min. | 2d4 | 2d3+2 |
| 3 | 5 | 3 min. | 3d4 | 3d3+3 |
| 4 | 6 | 4 min. | 4d4 (max) | 4d3+4 |
| +1 | +1 | +1 min. | - | +1d3+1 (no max) |
