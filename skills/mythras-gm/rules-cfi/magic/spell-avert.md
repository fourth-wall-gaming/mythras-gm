**Avert** (Abjuration)

- **Rank:** Cleric 0, Mage 0
- **Casting Time:** Instant
- **Duration:** Instant
- **Cost:** 1
- **Range:** 60 ft
- **Area:** 1 Spell
- **Resist:** None

Used to dismiss another Rank 0 spell within Range. Can be cast reactively to neutralize offensive Rank 0 spells using the Counter Magic Reactive Action.
