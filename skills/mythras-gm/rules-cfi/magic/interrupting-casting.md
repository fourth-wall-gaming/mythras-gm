**Multi-Turn casting:** Spending Action Points on Reactive Actions (Parry, Evade) during a multi-Turn cast delays completion but does not inherently disrupt it -- the caster simply finishes later.

**Full disruption** requires the caster to be injured or mentally subdued:

| Event | Effect |
| :-- | :-- |
| Spoil Spell Special Effect applied | Casting roll automatically Fails |
| Minor Wound | Willpower check: Failure makes the casting 1 Difficulty Grade harder |
| Serious Wound | Willpower check: Pass = 1 grade harder; Fail = 2 grades harder |
| Major Wound | Casting roll automatically Fails |
| Mental domination (succumbing) | Casting roll automatically Fails |
