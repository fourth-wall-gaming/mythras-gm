**Power Word - Stun** (Conjuration)

- **Rank:** Mage 3
- **Casting Time:** 1 Action
- **Duration:** See Below
- **Cost:** 3, +1/additional Intensity
- **Range:** 20 ft./Intensity
- **Area:** 1 Target
- **Resist:** Endurance

One target within line of sight and range who fails Endurance is stunned (briefly insensible; only Reactive defensive Actions allowed). Dispel Magic can cure early. Duration based on Rank or SIZ (whichever benefits target):

| Target Rank | Target SIZ | Duration |
| :-: | :-: | :-- |
| 0-1 | 1-20 | 4d4 Rounds |
| 2 | 21-40 | 2d4 Rounds |
| 3 | 41-60 | 1d4 Rounds |
| 4 | 61-80 | 1 Round |
| 5 | 81+ | Unaffected |
