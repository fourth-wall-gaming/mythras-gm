**Short Rest:** ~15 minutes of non-strenuous activity. No limit to Short Rests per day. Each player chooses one 15-minute Rest Action before moving to the next player.

**Rest Actions (each takes 15 minutes):**

| Action | Effect | Limit |
|:-------|:-------|:------|
| **Eat a Ration** | Remove any 1 Fatigue level no greater than Exhausted; also reduces hunger. If not greater than Winded, reduce 1 further Fatigue level (non-strenuous bonus). | Once per day |
| **Pray / Study spellbook** | Regain 1 Magic Point. If not greater than Winded, reduce 1 Fatigue level. | No limit |
| **Tend to wounds** | Use First Aid skill (1 action) or Healing skill (4 actions = 1 hour). | No limit |
| **Cast spells** | Cast any number of healing/buff spells on party members (limited by Magic Points). | No limit |
| **Prepare a spell** | After 8+ hours sleep: memorize or forget one Arcane or Divine spell (per memorizing rules). | No limit |
| **Dither (default)** | Waste 15 minutes. If not greater than Winded, reduce 1 Fatigue level. | No limit |
