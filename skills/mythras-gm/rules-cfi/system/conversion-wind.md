**Wind Table**

Wind Strength (STR) is expressed in miles per hour. Effects apply to physical skills (those using STR or DEX) and Movement Rate.

| MPH   | Type              | Effect on Skills | Effect on Movement |
|:-----:|:------------------|:----------------:|:------------------:|
| 0-10  | Calm              | None             | None               |
| 11-20 | Moderate Breeze   | None             | None               |
| 21-30 | Strong Breeze     | None             | Two-Thirds         |
| 31-40 | Light Gale        | Hard             | Two-Thirds         |
| 41-50 | Moderate Gale     | Hard             | Half               |
| 51-60 | Strong Gale       | Formidable       | Half               |
| 60+   | Storm/Hurricane   | Formidable       | One-Third          |

*Example: a Light Gale (STR 40) reduces human walking speed to two-thirds and skills are rolled at Hard difficulty.*

Light or moderate gale force winds shift Temperature Risk Rate one step cooler; strong gales and storms shift it two steps cooler (see Temperature Table).
