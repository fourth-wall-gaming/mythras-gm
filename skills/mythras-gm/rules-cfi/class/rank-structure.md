## Class Rank Structure

Classes are organized in a pyramidal hierarchy. **Rank 0** members are common members (most numerous). **Ranks 1-5** are professional members working full time under the class edicts. Numbers diminish at higher ranks.

## Gaining Rank

Rank is gained by meeting skill thresholds in Class skills (never counting skills that already had a +40 bonus, such as Customs or Native Language).

- **Rank 0**: Starting rank if prerequisites for Rank 1 are not met. No class Abilities.
- **Rank 1**: Any 5 Class skills at 40% or higher. ALL non-Ranked Abilities are acquired at no Experience Roll cost. Ranked Abilities may be purchased normally with Experience Rolls.
- Higher ranks: Meet the skill threshold shown on the class Rank Table (e.g., Rank 2 = any 5 at 70%; Rank 3 = any 4 at 90%; Rank 4 = any 3 at 110%; Rank 5 = any 2 at 130%).
- If the character is in good standing with any required Oaths and meets skill requirements, rank increase is automatic.

## What Each Rank Grants

Each class Rank Table lists:

- **Title**: the formal title at that rank.
- **Advancement**: number of Class skills and required skill level.
- **Spells in Memory** (casters only): how many spells of each rank may be memorized. Formula is (INT/4) + modifier per rank.
- **Hit Points**: certain ranks add +1 HP to ALL hit locations (cumulative).
- **Luck Points**: each rank above 1 grants +1 Luck Point (cumulative).
- **Action Points**: certain ranks grant +1 Action Point (cumulative).

## Ranked Abilities

Ranked Abilities may only be selected at or above the specified rank. Each costs a number of Experience Rolls equal to the rank (Rank 1 = 1 EXP roll; Rank 5 = 5 EXP rolls). No more than one Ranked Ability may be taken per session. Earlier rank Abilities remain available even after advancing to a higher rank.
